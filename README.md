# 题库刷题系统

一个基于 Web 的题库导入与刷题系统，支持从 Word 文档（.doc/.docx）导入题目，并提供在线刷题功能。

## 功能特性

- ✅ **题库导入**：支持 .doc 和 .docx 格式的题库文档导入
- ✅ **自动识别**：智能识别单选题和多选题，自动提取答案
- ✅ **题库管理**：查看、编辑、删除题目和题库
- ✅ **刷题练习**：随机抽题、答案验证、正确率统计
- ✅ **数据持久化**：题库数据保存在本地，重启后无需重新导入
- ✅ **可配置存储**：可自定义数据存储路径

## 快速开始

### 1. 安装依赖

```bash
cd backend
pip install -r requirements.txt
```

### 2. 启动服务

```bash
cd backend
python app.py
```

### 3. 访问系统

打开浏览器访问：http://localhost:5000

## 使用说明

### 导入题库

1. 点击左侧导航栏的"导入题库"
2. 将 .doc 或 .docx 文件拖入上传区域，或点击选择文件
3. 可选填写题库名称（留空则使用文件名）
4. 点击"开始导入"

### 题库格式要求

系统支持以下格式的题目文档：

```
第一章 章节名称

一、单项选择题
1、题目内容（ A ）
A 选项A    B 选项B
C 选项C    D 选项D

二、多项选择题
1、题目内容（ ABC ）
A 选项A
B 选项B
C 选项C
D 选项D
```

支持的答案格式：
- 中文括号：（A）、（ A ）、（ABC）
- 英文括号：(A)、( A )、(ABC)

### 刷题练习

1. 点击"开始刷题"
2. 选择题库、题型和题目数量
3. 点击"开始刷题"按钮
4. 选择答案后点击"提交答案"
5. 查看正确答案和解析
6. 完成后查看练习统计

### 系统设置

在"系统设置"页面可以：
- 修改数据存储路径
- 查看当前数据文件位置
- 清空所有数据

## 注意事项

1. 读取 .doc 文件需要系统安装 Microsoft Word
2. 建议使用 .docx 格式的文档以获得最佳兼容性
3. 题目格式需符合规范才能正确识别

## 技术栈

- **后端**：Python Flask
- **前端**：HTML5 + CSS3 + JavaScript
- **文档解析**：python-docx, pywin32
- **数据存储**：JSON 文件
